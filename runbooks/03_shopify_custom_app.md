# Runbook 03 — Shopify custom app install (for Peter)

**Time:** 10 minutes
**Who runs this:** Peter Dobias (or Matt with Peter screen-sharing)
**Outcome:** A `shpat_…` Admin API access token that n8n uses to pull orders, products, and customers from the Shopify store, plus webhooks pointing at n8n for real-time updates.

---

## Why we need this

Your Shopify store has years of order data and updates every few minutes. We're building a unified dashboard for One Eighty, and to do that the system needs **read-only** access to:
- Orders (no payment details, no card numbers)
- Products (titles, SKUs, prices, inventory)
- Customers (email, order history — for LTV / repeat-customer rate)

It does **not** need access to: app installations, store settings, themes, drafts, payment processing, or anything that can change your store.

You stay in control: revoke at any time by uninstalling the custom app.

---

## Step 1 — Open the custom app creator

1. Log in to Shopify admin (https://admin.shopify.com)
2. Go to **Settings** (bottom-left gear icon)
3. **Apps and sales channels** → **Develop apps for your store**
4. If you've never done this: **Allow custom app development** (one-time toggle)
5. **Create an app**
6. App name: `One Eighty Dashboard`
7. App developer: pick yourself
8. **Create app**

## Step 2 — Configure scopes (read-only)

1. **Configuration tab → Admin API integration → Configure**
2. Check ONLY these scopes:
   - [x] `read_orders`
   - [x] `read_all_orders` (lets us read orders older than 60 days — needed for backfill)
   - [x] `read_customers`
   - [x] `read_products`
   - [x] `read_inventory`
3. Leave everything else unchecked
4. **Save**

## Step 3 — Install the app

1. **API credentials tab → Install app → Install**
2. Confirm permissions
3. The page now shows **Admin API access token** — starts with `shpat_…`
4. **Click "Reveal token once"** and copy the entire string

⚠️ **IMPORTANT:** This token is shown ONLY ONCE. If you close the page, you'll need to uninstall + reinstall to get a new one.

## Step 4 — Send the token to Matt

**Use a secure channel:**
- Recommended: **1Password shared vault** (matej@oneeighty.cz already has access) or **Bitwarden Send** (one-time link, expires after 7 days or first view)
- DO NOT use: regular email, Slack DM, WhatsApp, or any chat platform

Also send:
- Your shop's `.myshopify.com` URL (e.g. `dr-dobias.myshopify.com`) — this is in your admin URL bar

## Step 5 — Webhooks (optional, set up by Matt later)

Matt will return with 5 webhook URLs (auto-generated by n8n) for real-time updates:
- `orders/create`
- `orders/paid`
- `orders/updated`
- `customers/update`
- `products/update`

You'll add these in **Settings → Notifications → Webhooks** (or Matt can add them via the Admin API using the token from Step 4).

## Step 6 — Klaviyo private API key (separate, also for Matt)

1. Klaviyo → **Account → API Keys → Private API Keys → Create Private API Key**
2. Name: `One Eighty Dashboard`
3. Permissions: **Custom Key**
   - Campaigns: **Read**
   - Flows: **Read**
   - Events: **Read**
   - Profiles: **Read**
   - Lists: **Read**
   - Metrics: **Read**
4. **Create**
5. Copy the key (starts with `pk_…`)
6. Send via the same secure channel as Step 4
